# Lemonade Stand Webshop (Next.js)

Dette projekt er en webshop udviklet i Next.js.

## Funktioner

- Produktliste (drinks fra API)
- Indkøbskurv (add/remove/quantity)
- Checkout side
- Profit beregner (lemonade stand)

## Teknologier

- Next.js
- React
- TypeScript
- Tailwind CSS

## Kør projektet lokalt

```bash
npm install
npm run dev
```
